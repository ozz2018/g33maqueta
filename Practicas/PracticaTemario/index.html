Creación de páginas web con HTML

Presentación

HTML (Hyper Text Markup Language) es un lenguaje para la creación de páginas web. La razón principal que motiva su uso es la creación de un código limpio, con la separación del contenido con respecto al diseño. El empleo de sus etiquetas semánticas permite mayor claridad en el desarrollo, el mantenimiento y el posicionamiento de las páginas en los buscadores. Además, HTML, en combinación con el lenguaje CSS 3 (Cascading Style Sheet), logra potenciar el aspecto visual de las páginas Web.

OBJETIVO

El participante identificará los elementos para el desarrollo de sitios web, que integrará con el lenguaje HTML y las hojas de estilo CSS de nivel 3.
